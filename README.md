# Tabuada JS

Projeto simples desenvolvido com HTML, CSS e JavaScript para praticar manipulação do DOM e eventos.

## Funcionalidades
- Gerar tabuada de um número
- Escolher multiplicador
- Limpar resultados

## Tecnologias
- HTML
- CSS
- JavaScript
